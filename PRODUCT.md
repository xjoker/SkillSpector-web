# Product

## Register

product

## Users

Developers and security reviewers evaluating AI agent skills before installation or release.
They need a fast, trustworthy way to inspect a skill bundle, zip archive, directory export, or
single file without reading every component manually.

## Product Purpose

SkillSpector detects malicious patterns, vulnerable instructions, risky scripts, and supply-chain
signals in agent skill packages. Success means a user can decide whether a skill is safe to install,
needs review, or should be rejected.

## Brand Personality

Calm, precise, security-minded.

## Anti-references

Avoid marketing-page decoration, oversized hero sections, novelty security visuals, and vague
"AI safety" gloss. The interface should feel like an inspection tool, not a landing page.

## Design Principles

- Put the scan workflow first; the first screen is the tool.
- Make risk level, score, and recommendation scannable before detail.
- Keep uploaded content transient and treat it as untrusted.
- Prefer dense, predictable controls over decorative composition.
- Surface errors with enough context to fix the input or configuration.

## Accessibility & Inclusion

Target WCAG AA contrast for text and controls. Use semantic form labels, keyboard-focusable
actions, reduced-motion-safe transitions, and color-independent severity labels.
